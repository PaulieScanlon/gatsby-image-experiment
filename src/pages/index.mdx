# withArtDirection

[withArtDirection](https://www.gatsbyjs.com/docs/reference/built-in-components/gatsby-plugin-image/#withartdirection) is one of the helper functions that ships with [gatsby-plugin-image](https://www.gatsbyjs.com/docs/reference/built-in-components/gatsby-plugin-image/)

_By default, the plugin displays different image resolutions at different screen sizes, but it also supports **art direction**, which is where a visually-different image is displayed at different sizes_
