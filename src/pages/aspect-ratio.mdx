# Aspect Ratio

It might sometimes be useful to display images of different aspect ratios on screens with a similar aspect ratio.

For example on larger screens that are normally _wider_ than they are _tall_ landscape images usually work best. However, on smaller screens that are _taller_ than they are _wide_ portrait images might be more suitable.

```javascript
const images = withArtDirection(getImage(nodes[0]), [
  {
    media: `(max-width: ${parseInt(theme.breakpoints[0], 10) - 1}px)`,
    image: getImage(nodes[1])
  }
]);
```
