# Printer

Using the `print` media query it's also possible to display a different image when the webpage is printed / is viewed in print preview.

```javascript
const images = withArtDirection(getImage(nodes[0]), [
  {
    media: `print`,
    image: getImage(nodes[1])
  }
]);
```
